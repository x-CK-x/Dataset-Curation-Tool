# Quick Queue Live Rows, Video Integrity Sampling, and Prediction Scores

v5.8.44 fixes Quick Tag queue visibility and score feedback while extending the integrity-classifier workflow to video.

## Quick Tag queue

The queue now shows every selected model immediately, even before the backend has returned all real job IDs.  This applies to download, load, unload, and inference queue actions.  The `Submit queue requests in parallel` option controls request submission order only; it does not change whether rows appear in the queue.

## Live status updates

The queue preserves temporary rows during live polling.  Each model row is patched as its own backend job changes state, so users do not have to wait for the whole queue to complete before seeing state changes.

## Prediction scores

Completed model inference jobs patch stored score rows directly into the active Tag Editor state.  Hover prediction bars should remain visible for tags generated by the completed model run.

## Video integrity checks

Nightshade/Glaze integrity classifier profiles can now run on videos by sampling frames.  Users can select sampling FPS, maximum frame count, PNG/JPEG/WebP frame output, compression/quality, and a highest-quality preset.  Scores are aggregated by label across sampled frames.
